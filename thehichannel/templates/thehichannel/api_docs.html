GET /api/user/[USER]

Returns posts from [USER].

Request:

$ curl https://thehichannel.com/api/user/stvnrlly/

Response: 

[
    {
        "hi": "hi",
        "sender": "stvnrlly",
        "timestamp": "2017-02-14T00:50:20.365314Z"
    },

POST /api/post/

Makes a post. Authorization is required: pass a valid token in the Authorization header.

Request:

$ curl -H "Authorization: token [YOUR_TOKEN]" https://thehichannel.com/api/post/

Response: 

{
    "hi": "hi"
}
